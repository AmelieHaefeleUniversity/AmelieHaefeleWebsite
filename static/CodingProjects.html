Here are my current Projects I've created as of Summer 2020

Tic Tac Toe Project

The project will let the user go first and place their O on an empty board. Then the board will be printed with where the machines X goes. The machine will not put an X on an already taken place the machine will also try and block and win itself sometimes. However to make it harder if the player puts their O on an already taken space they will lose a turn to keep them engaged. Then if either side wins the game will end and a message will be printed out telling the player wether or not they won.
